# Northwind SQL Business Analysis Project

## Overview
This project explores business insights using the Northwind dataset.  
The goal is to analyze customers, revenue, orders, employees, and product performance using SQL.

## Questions Answered
- Which countries have the most customers?
- Which customers generate the most revenue?
- Which products sell the most?
- Which employees handle the most orders?
- How can customers be segmented based on revenue?
- How can orders be categorized based on freight cost?

## Skills Demonstrated
- Joins
- Aggregations
- Group By
- Having
- Case statements
- Business logic analysis
- Data interpretation

## Tools Used
- PostgreSQL
- SQL

## Dataset
Northwind sample database
